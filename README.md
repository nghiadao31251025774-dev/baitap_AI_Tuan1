# Giải Bài Tập AI Trong Quản Trị Công Nghệ

Dự án này chứa các lời giải cho chuỗi bài tập chương 23 về ứng dụng Trí tuệ nhân tạo (AI) và dữ liệu địa không gian trong quản lý.

## 📌 Nội dung chính
* **Phân tích không gian:** Sử dụng Folium và GeoPy để trực quan hóa dữ liệu bản đồ.
* **Học máy (Machine Learning):** 
  * Dự báo nhu cầu bằng thuật toán Random Forest.
  * Phân cụm vị trí kho hàng tối ưu bằng K-Means.
* **Tối ưu hóa:** Tìm đường đi ngắn nhất (Dijkstra, A*) và tối ưu hóa tuyến giao hàng cho nhiều xe (VRP).
* **Mô phỏng:** Hệ thống điều phối xe thời gian thực và phân tích rủi ro tắc nghẽn.

## 🛠 Thư viện sử dụng
Để chạy được các file Notebook này, bạn cần cài đặt:
`pip install folium geopandas osmnx scikit-learn geopy networkx`

## 🚀 Dự án tiêu biểu
**Hệ thống đề xuất vị trí mở cửa hàng:** Phân tích mật độ khách hàng và vị trí đối thủ để tìm ra "điểm ngọt" kinh doanh tối ưu.
